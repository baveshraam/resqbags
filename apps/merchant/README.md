# ResQBag Merchant App

This is the independent merchant-facing application for ResQBag.

## How to run

1. Navigate to the `merchant-app` directory:
   ```bash
   cd merchant-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run start
   ```

## Notes

- **Design System**: This app intentionally copies the customer app's theme (colors, typography, spacing) to maintain brand consistency. It is designed to feel warm, food-first, and premium, avoiding a generic corporate SaaS look.
- **Backend/Auth**: Supabase and backend authentication are **not connected yet**. Auth screens are placeholders.
